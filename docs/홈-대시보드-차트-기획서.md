# AmInsight 홈 대시보드 차트 확장 기획서

> 참고: Google Analytics 4 보고서 개요 대시보드 구조 분석 기반
> 작성일: 2026-04-08

---

## 1. 현재 상태

현재 홈 대시보드에 있는 시각화:

| 구성요소           | 설명                                                   |
| ------------------ | ------------------------------------------------------ |
| KPI 카드 4개       | 광고비, 가입, 결제전환, ROAS (MoM 변화율)              |
| 성과 추이 차트     | 광고비/가입/결제/ROAS 탭 전환 Area Chart (일/주/월 봉) |
| 매체별 Bar Chart   | 매체별 광고비/결제/ROAS 탭 전환                        |
| 상세 데이터 테이블 | 전체 raw 데이터 정렬/필터/히트맵                       |
| AI 인사이트 패널   | 자동 분석 텍스트                                       |

## 2. GA 대시보드 분석 → AmInsight 매핑

GA 보고서 개요의 각 섹션을 AmInsight 광고 데이터 맥락으로 변환:

| GA 섹션                 | GA 내용                  | AmInsight 대응                                  | 우선순위 |
| ----------------------- | ------------------------ | ----------------------------------------------- | -------- |
| 활성 사용자 추이        | 라인 차트 (30일/7일/1일) | **이미 있음** (성과 추이 차트)                  | -        |
| 실시간 활성 사용자      | 현재 30분간 수치         | 최근 동기화 상태 + 오늘 실시간 지표             | P2       |
| 신규 사용자 채널 그룹별 | 가로 Bar Chart           | **매체별 가입수** 가로 Bar                      | P1       |
| 세션 채널 그룹별        | 채널별 세션 수 테이블    | **매체별 핵심 지표 요약 테이블**                | P1       |
| 국가별 활성 사용자      | 세계 지도 + 국가 테이블  | **국가별 광고비/ROAS 지도**                     | P1       |
| 시간 경과 사용자 활동   | 30일/7일/1일 비교 라인   | **기간 비교 미니 차트** (전월 vs 당월 오버레이) | P2       |
| 동질 집단 분석          | 주차별 리텐션 히트맵     | **주차별 성과 히트맵** (국가 × 주차)            | P2       |
| 페이지별 조회수         | 페이지 제목별 랭킹       | **작품별 성과 Top 10** 랭킹 테이블              | P1       |
| 이벤트 이름별 수        | 이벤트 카운트 테이블     | **목표별 전환 퍼널** (가입 → 결제)              | P2       |
| 항목별 구매 상품        | 상품명별 매출 테이블     | **작품별 결제금액 Top 10**                      | P1       |
| 플랫폼별 이벤트         | 도넛 차트                | **국가별 광고비 비중** 도넛 차트                | P1       |
| 채널별 신규 사용자 Bar  | 채널 그룹 가로 Bar       | **매체별 광고비 비중** 가로 Bar                 | P1       |

## 3. 추가할 차트/섹션 상세

### 3-1. 국가별 성과 요약 카드 (P1)

> GA의 "국가별 활성 사용자" 대응

```
┌──────────────────────────────────────────────────────────┐
│  국가별 성과 요약                                          │
│                                                          │
│  [세계 지도 — 국가별 색상 강도 = 광고비 비중]                │
│                                                          │
│  국가         광고비        ROAS      가입    결제전환      │
│  레진 KR     ₩4,090만    102.8%    8,092    4,196        │
│  US          ₩1,230만     87.5%    2,341    1,102        │
│  DE          ₩890만      145.2%    1,205      890        │
│  ...                                                     │
│                                                          │
│                              [탐색에서 자세히 보기 →]      │
└──────────────────────────────────────────────────────────┘
```

- **차트 타입**: 세계 지도(Choropleth) + 테이블
- **데이터**: 국가별 광고비, ROAS, 가입, 결제전환 집계
- **상호작용**: 국가 클릭 시 탐색 페이지로 이동 (해당 국가 필터 적용)
- **라이브러리**: `react-simple-maps` 또는 커스텀 SVG

### 3-2. 작품별 성과 Top 10 (P1)

> GA의 "페이지 제목별 조회수" + "항목별 구매 상품" 대응

```
┌──────────────────────────────────────────────┐
│  작품별 성과 Top 10                            │
│  [광고비 ▾] [가입 ▾] [결제 ▾] [ROAS ▾]  탭    │
│                                              │
│  #   작품명          광고비      가입   ROAS   │
│  1   렌탈걸즈        ₩520만     412  186.3%  │
│  2   하렘의남자      ₩480만     389  201.5%  │
│  3   징크스          ₩350만     298  145.2%  │
│  ...                                        │
│  10  보스의 반려묘    ₩85만      52   98.7%  │
│                                              │
│  [광고비 기준] 가로 Bar 차트                   │
│  ████████████████████  렌탈걸즈               │
│  ███████████████████   하렘의남자              │
│  ██████████████        징크스                 │
│                                              │
└──────────────────────────────────────────────┘
```

- **차트 타입**: 랭킹 테이블 + 가로 Bar Chart
- **데이터**: creative_name 기준 집계, 탭으로 정렬 기준 변경
- **포인트**: 어떤 작품에 돈을 많이 쓰고 있고, 어떤 작품이 효율이 좋은지 한눈에

### 3-3. 국가별/매체별 비중 도넛 차트 (P1)

> GA의 "플랫폼별 주요 이벤트" 대응

```
┌─────────────────────────────────────────────────────┐
│  광고비 비중                                          │
│  [국가별 ▾] [매체별 ▾]  토글                          │
│                                                     │
│      ┌───────┐                                      │
│     /  레진KR \    레진 KR    62.3%   ₩4,090만       │
│    |   62.3%  |    US        18.7%   ₩1,230만       │
│    |          |    DE        13.5%   ₩890만         │
│     \        /     기타       5.5%   ₩360만         │
│      └───────┘                                      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

- **차트 타입**: Donut Chart (Recharts Radial)
- **데이터**: 국가별 또는 매체별 광고비 비중
- **상호작용**: 토글로 국가↔매체 전환

### 3-4. 매체별 핵심 지표 요약 테이블 (P1)

> GA의 "세션 기본 채널 그룹별 세션수" 대응

```
┌────────────────────────────────────────────────────────┐
│  매체별 성과 요약                                        │
│                                                        │
│  매체        광고비       클릭      가입    ROAS   CTR   │
│  Meta       ₩3,200만   245,120   5,890  142.3%  2.8%  │
│  Google     ₩1,800만   189,450   3,210  118.5%  3.1%  │
│  YouTube    ₩780만      42,300     890   87.2%  1.2%  │
│  TikTok     ₩450만      38,900     670  165.8%  4.2%  │
│  Pinterest  ₩120만       8,200     180   92.1%  2.1%  │
│                                                        │
│  * 히트맵 컬러로 수치 강도 표시                           │
│                                                        │
└────────────────────────────────────────────────────────┘
```

- **차트 타입**: 히트맵 테이블 (기존 상세 데이터의 히트맵 기능 활용)
- **데이터**: 매체별 광고비, 클릭, 가입, ROAS, CTR 집계
- **포인트**: 기존 매체별 Bar Chart보다 한눈에 많은 지표 비교 가능

### 3-5. 기간 비교 미니 차트 (P2)

> GA의 "시간 경과에 따른 사용자 활동" (30일/7일/1일 라인 비교) 대응

```
┌──────────────────────────────────────────────┐
│  기간 비교                                     │
│  [당월 vs 전월] [당주 vs 전주]  토글            │
│                                              │
│  ── 당월 (3월)                                │
│  -- 전월 (2월)    점선 오버레이                 │
│                                              │
│   ₩2000만 ┤  ──────                          │
│   ₩1500만 ┤ /      \  ──                     │
│   ₩1000만 ┤/  - - -  \/  \                   │
│    ₩500만 ┤                                   │
│           └──1주──2주──3주──4주──              │
│                                              │

```

- **차트 타입**: 오버레이 라인 차트 (두 기간 겹침)
- **데이터**: 동일 기간 구간의 주차별/일별 지표
- **포인트**: 전월 대비 추세를 직관적으로 비교

## 4. 구현 우선순위

### Phase A (P1 — 즉시 구현 가능)

기존 데이터와 Recharts로 바로 만들 수 있는 것들:

| #   | 차트                         | 난이도 | 비고                                     |
| --- | ---------------------------- | ------ | ---------------------------------------- |
| 1   | 작품별 성과 Top 10           | 낮음   | 기존 집계 로직 재사용, 가로 Bar + 테이블 |
| 2   | 국가별/매체별 비중 도넛      | 낮음   | Recharts PieChart 활용                   |
| 3   | 매체별 핵심 지표 요약 테이블 | 낮음   | 기존 히트맵 기능 축소 적용               |
| 4   | 국가별 성과 요약 (테이블)    | 낮음   | 지도 없이 테이블 먼저                    |

### Phase B (P2 — 추가 작업 필요)

새 라이브러리 또는 계산 로직이 필요한 것들:

| #   | 차트                | 난이도 | 비고                            |
| --- | ------------------- | ------ | ------------------------------- |
| 5   | 기간 비교 미니 차트 | 중간   | 두 기간 데이터 fetch + 오버레이 |
| 6   | 주차별 국가 히트맵  | 중간   | 주차 집계 + 격자 렌더링         |
| 7   | 전환 퍼널           | 중간   | 단계별 전환율 계산              |
| 8   | 국가별 세계 지도    | 높음   | react-simple-maps 추가 필요     |

## 6. 기술 참고

- **차트 라이브러리**: Recharts 3.8 (이미 설치됨) — Area, Bar, Pie, Radial 모두 지원
- **데이터 소스**: 기존 `/api/dashboard` 응답의 AdRow[] 클라이언트 집계 (Phase A)
- **지도**: `react-simple-maps` (Phase B, 별도 설치 필요)
- **shadcn 규칙**: Card > CardHeader > CardContent > ChartContainer 구조 준수
- **차트 컬러**: `hsl(var(--chart-1))` ~ `hsl(var(--chart-5))` 시스템 사용
