# 탐색 추이 차트 기획서

## 배경

탐색(Explore) 페이지에 추이 차트가 없어 데이터를 시각적으로 파악하기 어려움.
실제 사용 플로우: 상세 보기 클릭 → 그래프(상단) + 데이터(하단) → 비교 필터로 기간 대비 확인.

## 요구사항

1. 결과 테이블 위에 추이 차트 표시
2. 가로축: 기간 (날짜/월) 고정
3. 세로축: 유저가 선택한 지표 (광고비, ROAS, CTR 등)
4. 비교 모드: 비교 주체별 색상 분리 (3월=파랑, 4월=빨강)
5. 미집계 구간: 0이 아닌 그래프 끊김 (null/undefined → connectNulls=false)

## 차트 표시 조건

- 차원에 `date` 또는 `month`가 포함된 경우에만 차트 표시
- 기간 차원이 없으면 차트 비노출 (테이블만)

## 구현 설계

### 컴포넌트: `ExploreChart`

### Props

```typescript
interface ExploreChartProps {
  rows: QueryResultRow[];
  dimensions: string[];
  metrics: string[];
  // 비교 모드
  compareResult?: CompareQueryResult;
  compareLabels?: { base: string; compare: string };
}
```

### 데이터 변환 로직

**일반 모드:**

- 가로축: `date` 또는 `month` 값
- 여러 차원이 있으면 (국가+날짜): 국가별 라인으로 분리
- 차원이 날짜만이면: 단일 라인

**비교 모드:**

- base/compare 두 결과를 같은 가로축에 겹침
- 가로축: 날짜 인덱스 (1일, 2일, ...)로 정규화
- base 라인 + compare 라인 각각 색상 분리

### 미집계 구간 처리

- Recharts `connectNulls={false}` → null 값에서 라인 끊김
- 데이터 없는 날짜는 값을 `undefined`로 설정 (0이 아님)

### 세로축 지표 선택

- 차트 상단에 지표 선택 드롭다운 (현재 쿼리 metrics 중 선택)
- 기본값: 첫 번째 지표
